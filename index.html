Common Queries

Is the application completely ad-free?

+

Yes. Dhvani does not inject any advertisements into audio queues or app screens. We pull clean, direct streams directly to the device interface.

How does offline caching work?

+

Downloaded and queued songs are managed locally using Hive local storage. Tracks are cached securely to prevent data corruption and allow offline listening.
